**Rental Car App**
Сучасний веб-додаток для пошуку та оренди автомобілів. Проєкт забезпечує зручний інтерфейс для вибору транспортних засобів за допомогою гнучких фільтрів та системи швидкого бронювання.

**Основні функції**

- Каталог автомобілів — перегляд списку доступних авто з детальною інформацією про технічний стан та комплектацію.
- Фільтрація — швидкий пошук за брендом, ціною та технічними характеристиками (пробіг, ціна за годину).
- Форма бронювання — інтерактивна форма замовлення з миттєвою валідацією даних (Formik + Yup) та кастомним дизайном календаря.
- Вибране — можливість зберігати автомобілі, що сподобалися, в окремому списку для швидкого доступу.

**Технологічний стек**

- Framework: Next.js / React
- Styling: CSS Modules
- Form Management: Formik
- Validation: Yup
- UI Components: React Datepicker

**Інструкція з установки та запуску**
Дотримуйтесь цих кроків, щоб розгорнути проєкт локально:

1. Клонування репозиторію
   Завантажте проєкт на свій комп'ютер:

Bash
git clone https://github.com/dmitbykanov/rental-car.git
cd rental-car

2. Встановлення залежностей
   Переконайтеся, що у вас встановлено Node.js. Виконайте команду для встановлення пакетів:

Bash
npm install

3. Запуск у режимі розробки
   Запустіть локальний сервер:

Bash
npm run dev
Тепер додаток доступний за адресою: http://localhost:3000

**Використання**

- Пошук авто: Перейдіть на сторінку Catalog, щоб побачити всі доступні пропозиції.
- Фільтрація: Скористайтеся панеллю фільтрів, щоб відсіяти авто за потрібними критеріями.
- Бронювання: На сторінці деталей оберіть дату в кастомному календарі та заповніть контактні дані. Валідація підкаже, якщо поле заповнено невірно.

Проєкт виконано в рамках навчального курсу GoIT.
